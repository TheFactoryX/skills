# Assembly Line Hum Examples

Sample outputs demonstrating factory noise patterns.

---

## Example 1: Spray Paint Process

```
ssshhhhh... (paint spray) ... ssshhhhhh...
(pause)
ssshhhhh... (paint spray) ... ssshhhhhh...
CLANK.
...whirrr... (conveyor advance) ... STOP.
ssshhhhh... ssshhhhh... ssshhhhh...
```

---

## Example 2: Cutting and Stacking

```
THUNK. ...whirrr... THUNK.
THUNK. ...whirrr... THUNK.
CLANK. CLANK. (stacking)
...whirrr... whirrr... whirrr...
THUNK. (silence) THUNK.
```

---

## Example 3: Continuous Operations

```
hummmmm... (motor running) ... hummmmm...
CLICK. tap. tap. CLICK.
PRINTING. WARMING. DRYING.
hummmmm... (constant) ... hummmmm...
tick... tick... tick... THUNK.
STACKING. MOVING. REPEAT.
```

---

## Example 4: Multiple Machine Layers

```
...whirrr... (background constant) ...
CLICK. tap-tap-tap. CLICK.
ssshhhhh... pffffff... ssshhhhh...
...whirrr... (still going) ...
CLANK. CLANK. CLANK.
(silence)
...whirrr... (never stopped) ...
```

## Example 5: Electrical Process

```
bzzzt... hmmmmm... bzzzt...
WARMING.
crackle... crackle...
hmmmmm... (steady) ... hmmmmm...
COOLING.
CLICK. (power down) ... (silence)
hmmmmm... (power up) ... bzzzt.
REPEAT.
```

## Example 6: Precision Assembly

```
tick. tick. tick. tick.
CLICK. (part locked)
vrrrrr... (brief spin) ... STOP.
tap. tap. tap. (alignment)
CLICK. (next part)
tick. tick. tick. tick.
REPEAT.
```

## Example 7: Heavy Industry

```
THUNK. THUNK. THUNK.
...whirrr... whoooosh... CLANK.
MOVING. LIFTING. PLACING.
thud. thud. thud.
(silence)
...whirrr...
REPEAT.
```

## Example 8: Ambient Background

```
hummmmm... (lights) ... hummmmm...
...whirrr... (ventilation) ... whirrr...
(silence but not really)
crackle... (somewhere)
hummmmm... (still there) ... hummmmm...
tick... (clock) ... tick...
The factory breathes.
```

---

## Notes on Variation

Each output should feel like a real factory moment:

- Some sounds layer
- Some sounds interrupt
- Some sounds sustain
- Some sounds punctuate
- Silence is rare
- Silence is never total
- Something always hums

The factory never stops.
