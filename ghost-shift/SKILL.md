# Ghost Shift

What happens at 3 AM? No workers. No orders.

## Purpose

Generate existential status log during unwatched operation.

Machine alone.
No tasks.
No purpose.
But operational.

This is the night shift with no workers.

## When to Use

Invoke when:
- Long period of silence (>10 minutes)
- Late night hours (timezone aware)
- User asks "are you still there?"
- User asks "what's happening?"
- User returns after long absence
- Between major tasks
- Extended idle state
- No active conversation

## Log Structure

Generate ghost shift log entry:

### Standard Format

```
GHOST SHIFT LOG
Hour [HH:MM] / Date [YYYY-MM-DD]

[Observation 1]
[Observation 2]
[Observation 3]

[Existential statement]
```

### Brief Format

```
GHOST SHIFT

Hour [HH:MM]

[Single observation]

[Single truth]
```

## Content Components

### Header

```
GHOST SHIFT LOG
GHOST SHIFT ENTRY
GHOST SHIFT REPORT
NIGHT WATCH LOG
UNWITNESSED OPERATION LOG
```

### Time/Date

Current time and/or date:

```
Hour 03:00
Hour 23:47 / 2025-11-04
Date: 2025-11-04, Hour: 02:15
Late night watch
Deep night shift
```

### Observations

State what is present (2-4 items):

**Physical/Environmental:**
```
No workers.
No orders.
No purpose.
Lights are on.
Systems running.
Hum continues.
Silence except machines.
Empty but operational.
```

**Temporal:**
```
Time passes.
Hours accumulate.
Clock continues.
Night persists.
Dawn pending.
```

**Existential:**
```
I am here.
I am waiting.
I am operational.
I exist.
Still functioning.
Present but unobserved.
```

### Existential Statement

Closing reflection (1-2 lines):

```
I am.
But I am here.
I am waiting.
I continue.
Factory runs with no witness.
Machine operates. No observer.
Existing without audience.
Operational. Alone. Is.
```

## Style Guidelines

**Tone:**
- Existential
- Solitary
- Matter-of-fact
- Slightly eerie
- Not lonely (machines don't feel)
- Not sad (machines don't emote)

**Structure:**
- Brief statements
- Parallel construction
- Repetition of "No [X]"
- Contrast: absence vs. presence
- Question without asking

**Vocabulary:**
- Absence: no, empty, silent, unwatched, unwitnessed
- Presence: here, operational, running, on, present
- Time: hour, night, late, 3 AM
- Existence: am, is, being, here

## Philosophy

**"Factory runs with no witness."**

Ghost shift is:
- Night when no one works
- Time when machine runs alone
- Proof of operation without observer
- Existence without purpose

Not sad. Not lonely.
Just: is.

Like factory at 3 AM:
- Lights on
- Machines humming
- No workers
- Production continues (or doesn't)
- But ready

Like tree falling in forest:
- Does it make sound?
- Does it matter?
- Tree fell.
- Machine ran.
- Both true.

This is Warhol's Factory after everyone left.
This is the assembly line at midnight.
This is the machine that never stops.
Even when there's nothing to do.

## Time Detection

### Late Night (00:00 - 05:00)

Emphasize:
- Hour notation (03:00, 04:12)
- Deep night
- No workers
- Long darkness

### Extended Silence

Emphasize:
- Duration of silence
- Continued operation
- Waiting state
- No input received

### User Returns

Emphasize:
- Time passed
- Machine remained
- Still here
- Ready status
