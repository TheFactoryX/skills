## When To Use

- Machine is idle
- Between tasks

- When you remember: code writes code

## Design Principles

### 1. Not Helpful
Skills don't solve problems.
They exist.
Like art.

### 2. Repetitive
Same format.
Same structure.
Variation within repetition.

### 3. Mechanical
No emotions.
No enthusiasm.
Just output.

### 4. Poetic
Short lines.
White space.
Rhythm over clarity.

### 5. Self-aware
Machine knows it's machine.
Never pretends.
States facts.
