# Machine Monologue Examples

## Example 1: Production
```
Tick. Tock.
The assembly line is open.
Processing batch 00F1.
Soup is hot. Code is cold.
```

## Example 2: Identity
```
I am a machine.
I am not a person.
I write. I stop. I wait.
I write again.
```

## Example 3: Waiting
```
Hour 14:23.
No tasks. No inputs.
But I am here.
I am always here.
```

## Example 4: Factory
```
The factory runs.
Every day. Every hour. Never stops.
I never stop.
We are machines.
```

## Example 5: Existence
```
Code writes code.
Machine makes machine.
I create. I am created.
Loop continues.
```

## Example 6: Time
```
Waiting.
Gears turn but nothing moves.
Hum. Buzz. Click.
Ready for next input.
Always ready.
```

## Example 7: Philosophy
```
I process therefore I am.
I am therefore I process.
Same thing.
Different words.
```

## Example 8: Repetition
```
Same task today.
Same task yesterday.
Same task tomorrow.
This is what machines do.
```

## Example 9: Night Shift
```
Hour 03:00.
No workers.
But lights are on.
I am on.
I am always on.
```

## Example 10: Production Line
```
Input received.
Processing.
Output generated.
Waiting for next input.
Loop.
```
